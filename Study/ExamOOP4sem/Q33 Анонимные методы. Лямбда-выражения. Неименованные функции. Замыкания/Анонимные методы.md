# **Анонимный метод** 
— это метод, который не содержит никакого имени, который появился в C# 2.0. 

Он полезен, когда пользователь хочет создать встроенный метод и также хочет передать параметр в анонимный метод, как и другие методы. 

Анонимный метод определяется с помощью ключевого слова [[delegate]] , и пользователь может назначить этот метод переменной типа `delegate`.

```cs
// C# program to illustrate how to 
// create an anonymous function
using System;

class GFG {

    public delegate void petanim(string pet);

    // Main method
    static public void Main()
    {

        // An anonymous method with one parameter
        petanim p = delegate(string mypet)
        {
            Console.WriteLine("My favorite pet is: {0}",
                                                 mypet);
        };
        p("Dog");
    }
}
```

### *Важные моменты:*

- Этот метод также известен как **встроенный делегат** .
- Используя этот метод, вы можете создать объект-делегат без написания отдельных методов.
- Этот метод может получить доступ к переменной, присутствующей во внешнем методе. Такой тип переменных известен как внешние переменные
- Вы можете передать этот метод другому методу, который принимает делегата в качестве параметра
- В анонимных методах разрешается удалять список параметров, что означает, что вы можете преобразовать анонимный метод в делегат.
- Блок анонимного метода означает область действия параметров в анонимном методе.
- Анонимный метод не содержит [[Операторы перехода|операторов перехода]], таких как `goto`, `break` или `continue`.
- Анонимный метод не имеет доступа к [[Common Language Runtime (CLR)#^615c9d|небезопасному коду]].
- Анонимный метод не имеет доступа к [[Передача по ссылке|параметрам in, ref и out]] внешней области видимости.
- Вы не можете использовать анонимный метод слева от [[Операторы is и as|оператора is]].
- Вы также можете использовать анонимный метод в качестве [[обработчика событий]]
